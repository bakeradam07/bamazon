# Bamazon

**Overview**

* Bamazon is a CLI App that is an Amazon-like storefront utilizing MySQL.  Bamazon interacts with the MySQL database allowing the user to view and purchase items.  Using MySQL, Javascript, and NodeJS Bamazon takes in the customers order, shows them the total for the order and then updates the database to remove the quantity the user chose from the stock of the product.

**Before you Begin**

* Install Node.js
* Install MySQL
* Using the bamazon.sql file, create a database in MySQL Workbench.
* Update the bamazon.js file to include your password for the database and then follow the instructions below for installing the necessary packages.
